# 停车场管理系统 java1028

本项目是一款基于前后端分离架构的停车场管理系统，采用 Spring Boot 和 Vue 技术栈开发。

## 技术栈

- 数据库：MySQL
- 开发语言：Java
- 前端技术：Vue、Element-UI
- 后端技术：Spring Boot、MyBatis Plus

## 功能模块

### 系统角色

#### 管理员用户

- 用户信息管理
- 车位信息管理
- 车位费用设置
- 停泊车辆管理
- 车辆进出管理
- 登录日志查询

#### 车主用户

- 个人中心
- 预定停车位
- 缴费信息
  - 停车缴费记录
  - 待缴费
  - 账户余额充值

## 运行环境

- 后端需安装 Java 环境以及 MySQL 数据库

## 部署步骤

1. 准备好 MySQL 数据库环境
2. 导入 SQL 文件创建数据库
3. 配置后端应用中的数据库连接信息
4. 部署前端至 Web 服务器
5. 部署后端至应用服务器

## 目录结构

```
parking_management_system
├── backend                # 后端代码
├── frontend               # 前端代码
└── sql                    # 数据库脚本
```

## 核心亮点

- 基于前后端分离架构，易于维护与扩展
- 采用流行的 Spring Boot 和 Vue 技术栈，保证了系统性能和开发效率
- 功能齐全，涵盖车位管理、车辆管理、缴费管理等停车场运营所需的核心功能
